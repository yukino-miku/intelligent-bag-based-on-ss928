# Embedded Contest Project

嵌赛长期协作项目资料夹。

## Directory Map

- `00_admin`: 比赛规则、时间线、分工、会议记录
- `01_requirements`: 需求、题目解读、评分点、约束
- `02_research`: 资料调研、往届方案、芯片/模块文档
- `03_design`: 总体方案、系统架构、接口定义、状态机、流程图
- `04_hardware`: 原理图、PCB、BOM、接线、外设模块
- `05_firmware`: MCU、FPGA、嵌入式代码
- `06_software`: 上位机、数据处理、工具脚本
- `07_tests`: 测试方案、测试数据、验收记录
- `08_media`: 图片、视频、演示素材
- `09_deliverables`: 论文、答辩 PPT、演示脚本、最终提交包
- `10_archive`: 废弃方案、旧版本、历史材料

## Collaboration Rule

每次讨论后，把关键决定、待办事项、风险和下一步记录到 `00_admin/project-log.md`。
